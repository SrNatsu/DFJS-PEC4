# Ejercicio 1

## A continuación, crea una carpeta ```PEC4_Ej_Teor```, dentro crea un fichero Markdown que tenga como nombre ```PEC4_Ej1_respuestas_teoria.md``` y explica qué realiza cada una de las siguientes opciones de Angular CLI.  

* **```ng new```:** Crea e inicializa una nueva aplicación de Angular.
  
* **```ng generate```:** Crea y/o modifica archivos basados en un esquema.
  * **```component```:** Crea un nuevo componente de Angular. Los componentes son los bloques de construcción básicos de una aplicación Angular. Cada componente consta de una clase TypeScript, una plantilla HTML y una hoja de estilos CSS opcional.
  
  * **```directive```:** Crea una nueva directiva en el proyecto. Una directiva se utiliza para ampliar el comportamiento o apariencia de los elementos del HTML y componentes. Las directivas permiten manipular el DOM, añadir atributos personalizados y responder a eventos.
  
  * **```enum```:** Crea una nueva enumeración en el proyecto. Las enumeraciones son una forma de definir un conjunto de constantes, haciendo el código más legible y mantenible.
  
  * **```guard```:** Crea un nuevo guardián de ruta. Los guardianes de ruta se utilizan para el control de acceso a partes de una aplicación comprobando ciertas condiciones antes de que se active la ruta.
  
  * **```interface```:** Crea una nueva interfaz en el proyecto. Una interfaz define la estructura de objetos en TypeScript, asegurando la seguridad del tipo y la claridad de código.
  
  * **```pipe```:** Crea una nueva tubería en el proyecto. Las tuberías se utilizan para transformar datos para que se muestren en las plantillas. Cogen los valores de entrada y les aplica una transformación específica.
  
  * **```service```:** Crea un nuevo servicio en el proyecto. Los servicios se utilizan para encapsular lógica reusable.

* **```ng serve```:** Construye y sirve la aplicación, reconstruyendo los archivos cambiados.

* **```ng test```:** Coge el nombre del proyecto, el especificado en la sección ```projects``` del fichero de configuración ```angular,json```.

* **```ng version```:** Muestra la versión de Angular CLI.
